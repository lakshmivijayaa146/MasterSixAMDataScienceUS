2 QN: why the name python ?
ANS: The Python programming language was named 
after the British comedy series Monty Python's Flying Circus, not the snake.
When Guido van Rossum, the creator of Python, 
began implementing the language in December 1989,
he was looking for a name that was short, unique, and slightly mysterious. 
He happened to be reading the published scripts from his favorite TV show, 
the 1970s BBC comedy Monty Python's Flying Circus, 
and decided that "Python" was the perfect working title for his new project

3 QN: why do we use react as frontend and python/java as backend ?
ANS: The reason we split development into 
React (frontend) and Python/Java (backend) is to 
separate the user experience from the complex business logic. 

This "separation of concerns" allows each part to do what it does best: 
React handles the visuals and interactivity, 
while Python or Java handles data, security, and heavy processing.

QN: ENV variables/Paths:
ANS: Python virtual env is a directory which will contain Pythoninterpreter,pip,Activation script,local site packages
Venv is package creating virtualenvironment.
command: python -m venv myvenv

Path setting up in windows :

If you forgot to check the "Add Python to PATH" box during installation, you can manually add it to your Windows Environment Variables to run python from any terminal. 
1. Locate Your Python Installation Path
Before changing settings, you need the exact address of your Python executable. 
Open the Start Menu, type "Python", right-click the app, and select Open file location.
Right-click the shortcut in that folder and select Open file location again to reach the folder containing python.exe.
Copy the path from the address bar (e.g., C:\Users\<User>\AppData\Local\Programs\Python\Python312). 
2. Add to Environment Variables
Press the Windows Key, type "Edit the system environment variables", and open it.
Click the Environment Variables button at the bottom.
Under User variables, find the Path variable and click Edit.
Click New and paste the Python path you copied in Step 1.
Click New again and paste the same path but add \Scripts at the end (e.g., ...\Python312\Scripts) to enable tools like pip.
Click OK on all windows to save. 
3. Verify the Path
Close any open Command Prompts or PowerShell windows and open a new one.
Type python --version to see your installed version.
Type where python to see exactly which path Windows is currently using.

QN: What is Floor and Ceil ?
ANS: math.floor(x): Rounds down to the largest integer less than or equal to x.
math.ceil(x): Rounds up to the smallest integer greater than or equal to x.

Example: 
import math

val = 3.7

print(math.floor(val))  # Output: 3
print(math.ceil(val))   # Output: 4

# Negative numbers round toward infinity
neg_val = -3.7
print(math.floor(neg_val)) # Output: -4
print(math.ceil(neg_val))  # Output: -3
