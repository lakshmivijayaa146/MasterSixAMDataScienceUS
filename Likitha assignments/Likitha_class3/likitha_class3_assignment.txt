1. Difference between Compilation and Interpretation

Compilation: The whole program is converted into machine code before execution by a compiler.
Example languages: C, C++

Interpretation: The program is executed line by line by an interpreter during runtime.
Example languages: Python, JavaScript

2. Is Python Compiled or Interpreted?

Python is both compiled and interpreted.

Process:

Python code is first compiled into bytecode.

The Python Virtual Machine (PVM) then interprets and executes the bytecode.

Short interview answer:

Python first compiles code into bytecode and then interprets it using the Python Virtual Machine.
3.
Python is dynamically typed, so a variable can change its type during execution.

Code:

a = 10
a = 'liki'
a = 10

What happens:

a = 10 → a stores an integer

a = 'liki' → a now stores a string

a = 10 → a again stores an integer

Python simply reassigns the variable each time, and its type changes based on the value.
