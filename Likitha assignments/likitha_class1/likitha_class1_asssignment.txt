1. HYPER-PERSONALIZATION

2. WHITE CODING IN AI
White coding means building an AI solution from scratch without using pre-built models or frameworks.

Example:
Instead of using a sentiment analysis library, you write logic yourself.

Example Python:

text = "This movie is good"

if "good" in text:
    print("Positive sentiment")


3. CODE COVERAGE
Code coverage tells how much of your program is tested by test cases.

Example:

def check_number(n):
    if n > 0:
        return "Positive"
    else:
        return "Negative"

If your tests only check positive numbers, the negative condition is never tested.  
So coverage is incomplete.

7. OOP (OBJECT ORIENTED PROGRAMMING) CONCEPTS

Class
A class is a blueprint for creating objects.

Example:
class Car:
    color = "red"


Object
An object is an instance of a class.

Example:
my_car = Car()
print(my_car.color)


Encapsulation
Encapsulation hides internal details of a class.

Example:
class Bank:
    def __init__(self):
        self.__balance = 1000


Inheritance
Inheritance allows a class to reuse properties from another class.

Example:

class Animal:
    def speak(self):
        print("Animal sound")

class Dog(Animal):
    pass

d = Dog()
d.speak()


Polymorphism
Same method name but different behavior.

Example:

class Dog:
    def sound(self):
        print("Bark")

class Cat:
    def sound(self):
        print("Meow")

animals = [Dog(), Cat()]

for a in animals:
    a.sound()

9. MEAN STACK

10. MERN STACK
MERN is similar to MEAN but uses React instead of Angular.

M – MongoDB
