 GITLAB AND BITBUCKET
GitLab and Bitbucket are platforms for storing and managing Git repositories.

They allow developers to:
- store code
- collaborate
- manage version control
- run CI/CD pipelines.


2. FRAMEWORK VS PROGRAMMING LANGUAGE

Programming Language:
A language used to write programs.

Examples:
Python
Java
JavaScript

Framework:
A tool built using a language that helps build applications faster.

Examples:
Django (Python)
Spring (Java)
React (JavaScript)


3. OPEN SOURCE VS CLOSED SOURCE

Open Source:
Source code is publicly available.

Examples:
Linux
Python
Apache

Closed Source:
Source code is private.

Examples:
Microsoft Windows
Adobe Photoshop


4. UNIT TESTING
Unit testing means testing individual parts of code.

Example:

def add(a,b):
    return a+b

print(add(2,3))

Expected output: 5


5. TDD (TEST DRIVEN DEVELOPMENT)
TDD means writing tests before writing the actual code.

Steps:
1. Write test
2. Write code
3. Refactor

8. MCP (MODEL CONTEXT PROTOCOL)

MCP allows AI models to interact with external tools and data sources while maintaining context.

Example uses:
- AI accessing company documents
- AI querying a database
- AI calling APIs

It helps AI systems connect with real-world tools.
